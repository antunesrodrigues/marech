# Directory Structure – Marech

## Default
  This is the default directory structure. Do not panic, you can change whatever you want.
  ```
  .
  ├── src
  │   ├── marech
  │   │   └── foo.html
  │   │
  │   └── whatever.html
  │ 
  ├── dist
  │   └── whatever.html
  │
  └── marech-config.json
  ```
