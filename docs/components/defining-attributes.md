## Defining Attributes
- ### Import
  Set the attributes the same way you define for HTML tags.
  ```html
  <Marech@imgs fileName="cool" />
  ```

- ### Component
  You can receive the attributes in this way:
  ```html
  <img src="{fileName}.png">
  ```
