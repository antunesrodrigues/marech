## O que o projeto faz

## 🧷 How to Install
1.  Install and Start
  ```sh
  # Install Marech CLI globally
  npm i marech -g

  # Start marech 
  marech init
  ```

## 🏹 Sample code
```sh
marech compile
```

## 🔥 Como rodar localmente
Clone this repo
```
git clone https://github.com/antunesrodrigues/marech.git
```

At the folder, link this package
```
cd marech
npm link
```
